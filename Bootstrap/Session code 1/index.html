Example 1: Basic Bootstrap Setup

Getting Started with Bootstrap

Bootstrap makes responsive web development faster and easier. This framework provides pre-built components and a powerful grid system.

Key Concept: Bootstrap uses a mobile-first approach, meaning designs work on small screens first, then scale up to larger screens.
